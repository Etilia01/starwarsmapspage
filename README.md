A small website I made as a prop for my current SW5E DnD campaign!

Currently unlocked/available is a map and very little information about the planet Dathomir, which is accesible by typing o:6 (the quadrant in which Dathomir is) in the search bar and clicking the search icon! The players havent discovered much about this planet (or any really) yet, so there isnt much information, but I will be updating the info-boxes as they progress :)

They will also only be able to access information about planets they have visited while already in posession of the datapad that this website is supposed to be, which currently is only Dathomir, so thats the only one you can see so far... But once theyre done looking around the underground structure there theyll probably go on to Mustafar (if everything goes as planned :3) so I will be working on a page for that one at some point...
